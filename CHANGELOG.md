# 更新日志 (Changelog)

本文件记录 iRAS 的所有显著变更。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.0.0] - 2026-04-17

### 首次发布

#### 核心计算
- 多阶段养殖规模反推（尾数流守恒 + Little's Law）
- 精确循环方程水质稳态仿真：C_out = Δ/(1-R)，闭合验证误差 = 0
- O₂ 按位置分配：鱼池主增氧 + BF 独立粗孔曝气，不重叠
- 盐度体系：Benson & Krause (1984) DO 饱和度 + USGS 盐度修正 + 硝化效率修正
- 碱度收支：硝化消耗 7.14 g CaCO₃/g TAN-N + 反硝化回收 3.57 + NaHCO₃ 投加
- CO₂ 稳态质量守恒公式
- 热泵 + 板式换热回收能耗计算

#### 水处理工艺
- 主流串联：RDF 微滤 → 生物滤池 (MBBR) → UV 消毒 → 增氧/脱气
- 旁路建模：蛋白分离器（空气/臭氧）、AOP 高级氧化、反硝化反应器
- 蛋分旁路模型：系统等效去除率 = 流量比 × 单次通过效率
- VTR/VDR 温度修正（Arrhenius）+ 盐度修正

#### 设备与成本
- 设备规格自动选型：RDF/BF/UV/蛋分/AOP/反硝化/增氧/脱气/泵/风机/热泵
- 泵功率分项计算：主循环泵 + 5 条支路泵 + RDF 反冲泵
- 运行成本：峰值 + 日均双口径
- 投资概算：15 类设备单价可编辑 × 工程放大系数，标注 ±40% 精度
